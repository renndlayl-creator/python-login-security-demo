# Python Login Security Demo

Proyecto educativo desarrollado en Python para practicar conceptos básicos de autenticación y seguridad informática.

## Características

- Interfaz gráfica con Tkinter
- Sistema de inicio de sesión
- Límite de intentos fallidos
- Bloqueo temporal después de varios intentos incorrectos
- Registro de eventos en archivo de logs
- Terminal visual estilo "cybersecurity"
- Simulación de IP para fines educativos

## Tecnologías utilizadas

- Python 3
- Tkinter
- Datetime
- Time
